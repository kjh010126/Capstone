# Capstone

캡스톤 디자인 수업용

## 프로젝트 개요
실습실 PC 사용 관리 시스템은 학생이 **모바일 앱에서 로그인**한 뒤, 모바일 앱에 표시된 6자리 인증 코드를 **PC 앱의 코드 인증 화면에 입력**하여 PC 사용을 인증하는 시스템이다.

인증이 완료되면 학생은 PC 앱과 모바일 앱에서 사용 시간과 종료 예정 시간을 확인할 수 있고, 필요할 경우 사용 시간 연장 신청을 할 수 있다. 연장 신청이 없으면 세션은 자동 로그아웃되며, PC 앱은 종료 시각에 운영체제 종료 명령을 실행하여 자동 종료를 수행한다.

관리자는 PC 앱에서 관리자 계정으로 로그인하여 회원 정보, 현재 사용 세션, 세션 히스토리를 조회하고 악성 사용자 제한 및 특정 PC 강제 종료를 수행할 수 있다.

## 팀원 역할
- 재형(나): PC 앱 / React
- 현래: 모바일 앱 / React
- 우혁(팀장): 서버 / DB
  - DB: PostgreSQL
  - 서버: Go

## 핵심 사용 흐름
1. 학생이 모바일 앱에서 로그인한다.
2. 모바일 앱은 서버에 인증 코드 발급을 요청하고, 3분 유효시간의 6자리 코드를 표시한다.
3. 학생은 PC 앱의 코드 인증 화면에서 해당 코드를 입력한다.
4. 서버가 코드를 검증하고 사용자-PC 세션을 생성한다.
5. PC 앱은 사용 가능 상태로 전환되며 사용 시간, 종료 예정 시간, 알림을 표시한다.
6. 학생은 모바일 앱 또는 PC 앱에서 사용 시간 연장 신청을 할 수 있다.
7. 연장 신청을 하지 않으면 세션은 자동 로그아웃된다.
8. PC 앱은 종료 시각에 도달하면 운영체제 종료 명령을 실행하여 자동 종료를 수행한다.
9. 관리자는 PC 앱에서 사용자 조회, 세션 조회, 사용자 제한, 강제 종료를 수행한다.

## 화면 구조

### 모바일 앱

#### 로그인 전 화면
- 로그인
- 회원가입
- 학번 찾기
- 비밀번호 찾기

#### 홈 화면
- PC 인증 코드 표시
- 사용 시간 확인
- 사용 시간 연장 신청
- 알림 확인
- 비밀번호 변경
- 회원탈퇴
- 로그아웃

### PC 앱

#### 코드 인증 화면
- 모바일 앱 인증 코드 입력
- 관리자 계정 접속

#### 일반 사용자 메인 화면
- 사용 시간 확인
- 종료 예정 시간 확인
- 사용 시간 연장 신청
- 알림 확인
- 연장 신청 미수행 시 자동 로그아웃
- 종료 시 자동 종료

#### 관리자 로그인 화면
- 관리자 로그인

#### 관리자용 화면
- 사용자 리스트 표시
- 선택한 사용자의 현재 세션 정보 표시
- 선택한 사용자의 세션 히스토리 표시
- 악성 사용자 제한
- 특정 PC 강제 종료 명령

## 현재 반영된 핵심 변경사항
- 일반 사용자의 **PC 앱 로그인 기능 제거**
- 인증 흐름을 **PC 앱 표시 → 모바일 입력** 방식에서 **모바일 표시 → PC 앱 입력** 방식으로 변경
- **연장 신청을 하지 않으면 자동 로그아웃**되도록 정책 변경
- 관리자 기능은 기존처럼 **PC 앱 중심**으로 유지
